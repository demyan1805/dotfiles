## tmux
`CMD-S-[` `]` – switch window

`CMD-S-P` – switch to last window

`CMD-S-Left` `Right` – move window

`CMD-S-\` – window list view

`CMD-T` – new window

`CMD-W` – close window

`CMD-F` – copy mode

`CMD-K` – (Alacritty) send k (clear)

`CMD-R` – send tmux reset command
